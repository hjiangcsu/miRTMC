This folder include dataset D1 in paper " miRTMC: A miRNA target prediction method based on matrix completion algorithm",
which is submitted to the Journal of Biomedical and Health Informatics
sim_g.mat: gene similarity matrix   (Stored with Git LFS)
sim_m.mat: miRNA similarity matrix
tmMat.mat: adjacency matrix of miRNA-gene interaction bipartite graph.

If you can't download file "sim_g.mat", please download it here : http://bioinformatics.csu.edu.cn/resources/softs/jianghui/miRTMC/dataset_D1.rar
